AspNet Core MVC Web application secured by Identity Server 4 - Implicit Flow
1.Clone Repo
2.Run >dotnet restore
3.Run >dotnet build
You should be running on port http://localhost:5002

Depends on having an Identity Server 4 properly configured and running @ http://localhost:5000
Identity server is in the following repository: https://github.com/horaciodev/IdSvr4POC   (dev branch)
